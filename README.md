# PyGetGPT
使用python构建的简易语言模型api调用系统
使用web页面作为GUI易于部署
---
## 使用的外部库
- flask
- flask_cors
- zhipuai
- pymysql
- requests
- openai

## 使用pip安装依赖
~~~ bash 
pip install pymysql requests flask zhipuai openai
pip install pymysql requests flask zhipuai openai -i https://pypi.tuna.tsinghua.edu.cn/simple
~~~
